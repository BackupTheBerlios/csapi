Documents for Developers

Some documents are available to help developers to understand and reuse both the csapicore Java API and the csplugin Eclipse plugin. All documents can be found on the csapi.berlios.de site, section Documentation.

  • Exploring the Change/Synergy Perl API protocol ( PDF file) is a description of the protocol used by the Perl API when communicating with the Synergy/Change server.
  • CSAPI Core classes Architecture ( PDF File not available). An in-depth view of the csapicore architecture. UML diagrams, conceptual comments.

Documents for Administrators

The csapi project is also the home of an introduction to Synergy/Change administration document. It provides remarks and observations, with some bits of knowledge which you may interested in. Don't hesitate to contribute, too.
