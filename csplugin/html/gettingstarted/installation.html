Installation of the CSAPI plugin

The preferred way to install the csapi plugin is through the update manager. It keeps a track of every software installed, and allows to come back to older versions if needed.

Update manager

When the update manager is launched, click on "search for new features to install" add a new remote site with

  • Name = "CSAPI plugin" and
  • URL = "http://csapi.berlios.de/updates".

From zip files

Get the last zip files from the official download site:

CSAPI download on berlios.de

Download both csplugin and csapicore, whatever version, and unzip it. A directory "plugins" should be created, copy files from it to the Eclipse plugins directory.

Restart Eclipse.
